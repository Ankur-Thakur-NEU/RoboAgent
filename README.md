# RoboAgent

Brief description of the RoboAgent project.

## For Designers
1. Export your model as `.glb`.
2. Drop it into `starbots-site/src/assets/models/`.
3. Update the path in `starbots-site/src/config/assets.js`.
4. Run `npm run dev` from `starbots-site/` and refresh the page.

Notes:
- Barista model: update `ASSET_PATHS.baristaModel`.
- Customizer preview model: update `ASSET_PATHS.customizerModel`.
- Texture swaps: update `ASSET_PATHS.coffeeBeanTexture`.
- Shader swaps: update `ASSET_SHADERS` in `starbots-site/src/config/assets.js`.
